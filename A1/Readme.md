# Assignment 1 4DV806

## About

This contains assignment 1 for course 4DV806. In this assignment we visualize the publication stats data for Pendelton State University (PSU).

## Tech Stack
Following are the tech stack used:
- HTML, CSS
- JavaScript
- D3.js for visualization(used from CDN)

## How to Run

To run the application, execute the following command:

```bash
python3 -m http.server 8000
```

The app should be available at [http://localhost:8000](http://localhost:8000)
